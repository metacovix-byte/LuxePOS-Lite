  • Saisie produits + photos haute qualité (~200 KB max)
  • Import Excel intelligent (11 feuilles supportées)
  • 3 emplacements : Atelier · Annemasse · Salon Genève
  • Toggle Vendu/Pas vendu en 1 click
  • Clients + Historique d'activité
  • Backup OneDrive chiffré + Export Excel
  • Mode offline + queue de synchro
  • Page Feedback pour remonter avis/bugs
